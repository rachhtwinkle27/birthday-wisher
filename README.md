# Birthday Wisher
An automated Python application that sends personalized birthday emails using Gmail SMTP.

The project reads birthday information from a CSV file, checks if today matches anyone's birthday, selects a random birthday letter template, personalizes it with the recipient's name, and sends the email automatically using GitHub Actions.

## Features

- Reads birthday data from a CSV file
- Checks today's date automatically
- Chooses a random birthday letter
- Replaces placeholder text with the recipient's name
- Sends email using Gmail SMTP
- Runs automatically everyday using GitHub Actions
- Stores email credentials securely using GitHub Secrets

## Technologies Used

- Python
- Pandas
- SMTP
- GitHub Actions
- GitHub Secrets
- CSV
